# Changelog

## [1.0.0] - 2025-01-05

### Added
- Initial release
- Centrifuge: First-stage isotope separation (0.5x risk)
  - 40% concentrated Atlantum output
  - 50% waste output
  - Max 2 safety modules
- Enrichment Chamber: Main enrichment process (1.5x risk)
  - 20% enriched Atlantum output (valuable!)
  - 70% waste output
  - Containment system that degrades during operation
  - Containment breach causes radiation leaks
  - Max 4 safety modules
- Waste Processor: Safe waste handling
  - 80% depleted Atlantum (safe)
  - 5% recovered material
- Radiation Zones
  - Created by explosions and breaches
  - Damages nearby facilities
  - Decays over time
- Safety Module System
  - Each module reduces explosion risk by 10%
  - Visual indicators show installed modules
- Explosion Effects
  - Particle effects and light flash
  - Chain reaction potential
- BioProcessing Integration
  - Bio-remediation accelerates radiation cleanup
- Configurable risk levels, damage, and decay times
